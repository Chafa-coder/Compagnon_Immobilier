# Projet Streamlit - Analyse et Prédiction des Prix Immobiliers

## Description
Application Streamlit pour :
- Exploration et nettoyage des données
- Préparation des features
- Modélisation des prix
- Simulation et évolution dans le temps et la géographie
- Prédiction interactive

## Structure du projet
- `Overview_projet.py` : page d'accueil
- `pages/` : pages Streamlit complémentaires
- `data/` : dossiers inputs et outputs
- `requirements.txt` : dépendances Python

## Déploiement
1. Installer Python 3.11
2. Installer les dépendances :
```bash
pip install -r requirements.txt
